Library yang digunakan:
Membutuhkan library pandas.

Algorithm:
Code akan menimport iris.csv sebagai data acuan, dan mengambil nilai min dan max dari setiap kategori iris.
User diminta memasukkan nilai kriteria untuk melakukan prediksi.
Jika nilai yang dimasukkan user berada diantara min dan max dari suatu kategori, nilai sesuai + 1, dan langkah ini dilakukan untuk setiap kategori.
Nilai presentasi prediksi ditampilkan dengan membandingkan nilai sesuai setiap kategori dengan jumlah kriteria.
Nilai max dari nilai sesuai akan ditampilkan sebagai hasil prediksi.

Komentar:
Algoritma ini mungkin berfungsi untuk dataset yang memiliki nilai integer atau float, namun untuk dataset ber tipe data string sepertinya tidak bisa.
