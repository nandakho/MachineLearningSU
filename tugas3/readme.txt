Library yang digunakan:
Membutuhkan library pandas.

Algorithm:
Code akan menimport data.csv sebagai data pembelajaran menggunakan algoritma Naive Bayes.
User diminta memasukkan nilai kriteria untuk melakukan prediksi/uji coba.
Output yang dihasilkan merupakan keputusan berdasarkan algoritma naive bayes berdasarkan training dari data.csv.

Sumber Data:
Data diambil dari https://www.kaggle.com/piyushgoyal443/red-wine-dataset
Dengan sedikit data cleaning (penghapusan kolom nomor urut)

Komentar:
Algoritma ini mungkin berfungsi untuk dataset yang memiliki nilai integer atau float, namun untuk dataset ber tipe data string sepertinya tidak bisa.
